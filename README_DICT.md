# Install

 1. Copy "Erlang OTP Reference.dictionary" to "~/Library/Dictionaries".
 2. Launch "Dictionary.app".
 3. Open setting and check Erlang Dictionary.

# Command line tool

 * erlre/erlre
 * erlre_rb/erlre.rb (macruby)

    % erlre foldl
    % erlre lists:foldl/3
    % erlre gen_server
